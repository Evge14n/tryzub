## Quick Start

```bash
# Встановлення
git clone https://github.com/Evge14n/tryzub.git
cd tryzub
cargo install --path .

# Перша програма
echo 'друк("Привіт, Тризуб! 🇺🇦")' > привіт.тризуб
тризуб запустити привіт.тризуб
```

## Синтаксис

```
змінна ім_я = "Тризуб"
змінна версія = 8

функція привітати(хто) {
    друк("Привіт, " + хто + "!")
}

привітати(ім_я)
```

## Можливості

| Фіча | Опис |
|-------|------|
| Змінні та типи | `змінна`, `стала`, type inference |
| Функції | параметри за замовчуванням, генерики `<Т>` |
| Структури | `структура Точка { х, у }` |
| Трейти | `трейт`, `реалізація`, operator overloading |
| Pattern matching | `зіставити`, guards, exhaustiveness |
| Модулі | `імпорт`, namespace ізоляція, stdlib |
| Ітератори | ліниві Range, pipeline `\|>`, `.фільтрувати()` |
| Async | `чекати`, `все()`, `перегони()`, `потік()` |
| Веб-сервер | маршрути, шаблони, сесії, JWT auth |
| База даних | SQLite ORM: `бд_створити`, `бд_знайти` |
| Тип-система | анотації + auto-inference |
| FFI | `зовнішня_бібліотека`, `зовнішній_виклик` |
| JIT | x86_64 JIT компілятор |
| Системне | inline asm, raw memory, bootable kernel |

## Інструменти

```bash
тризуб запустити файл.тризуб      # Запустити програму
тризуб інтерактив                  # REPL з історією
тризуб тестувати файл.тризуб      # Запустити тести
тризуб формат файл.тризуб         # Автоформатування
тризуб лінт файл.тризуб           # Перевірка стилю (AST)
тризуб док ./src --вивід ./docs   # Генерація документації
тризуб lsp                        # LSP сервер для IDE
тризуб новий проект               # Створити проект
тризуб встановити <git-url>       # Встановити пакет
```

## Приклади

### Fibonacci
```
функція фіб(n) {
    якщо (n <= 1) { повернути n }
    повернути фіб(n - 1) + фіб(n - 2)
}

для (і в 0..10) { друк(фіб(і)) }
```

### Веб-сервер
```
веб_отримати("/", |запит| {
    веб_html("<h1>Привіт з Тризуб!</h1>")
})
веб_запустити(3000)
```

### Трейти та operator overloading
```
структура Вектор { х, у }

реалізація Вектор {
    функція додати(себе, інший) {
        повернути Вектор { х: себе.х + інший.х, у: себе.у + інший.у }
    }
}

змінна а = Вектор { х: 1, у: 2 }
змінна б = Вектор { х: 3, у: 4 }
змінна с = а + б   // Вектор { х: 4, у: 6 }
```
