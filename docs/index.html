Мова Тризуб

Сучасна, безпечна, компільована мова програмування з повністю українським синтаксисом. Pattern matching, трейти, pipeline, LLVM-компіляція.

curl -sSL https://evge14n.github.io/tryzub/install.sh | sh

Можливості

Все найкраще з сучасних мов — одним пакетом, рідною мовою

🧩

Алгебраїчні типи

Enum-типи з даними, Опція<Т> та Результат<Т, П> для безпечної обробки помилок замість null та виключень

🎯

Pattern Matching

Зіставлення зразків з деструктуризацією, guard-умовами та вкладеними зразками для елегантного коду

🔗

Трейти

Типажі з методами за замовчуванням — поліморфізм без складності наслідування класів

🔀

Pipeline оператор

Оператор |> для побудови ланцюжків перетворень — читабельно, як конвеєр даних

⚡

LLVM + Cranelift

Три бекенди: нативна компіляція через LLVM, JIT через Cranelift, інтерпретація через VM

🇺🇦

Українська

Всі ключові слова, повідомлення помилок та документація — рідною мовою

// Алгебраїчний тип
тип Фігура {
    Коло(дрб64),
    Прямокутник(дрб64, дрб64)
}

функція площа(ф: Фігура) -> дрб64 {
    зіставити ф {
        Коло(р) => 3.14 * р ** 2,
        Прямокутник(ш, в) => ш * в
    }
}

функція головна() {
    змінна р = площа(Коло(5.0))
    друк(ф"Площа: {р}")
}
