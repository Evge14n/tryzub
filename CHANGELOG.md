# Історія змін — Тризуб

## [3.9.0] - 2026-03-26

### Нові можливості
- **Файловий I/O**: `файл_прочитати`, `файл_записати`, `файл_існує`, `файл_рядки`, `файл_додати` — реальні операції через std::fs
- **Генератори**: `віддати` складає значення, `.в_масив()`, `.взяти(n)`, `.наступний()`, for-in по генератору
- **Extension methods цл64**: `.парне()`, `.непарне()`, `.абс()`, `.в_текст()`, `.в_дробове()`, `.степінь()`, `.мін()`, `.макс()`
- **Extension methods дрб64**: `.абс()`, `.округлити()`, `.підлога()`, `.стеля()`, `.в_текст()`, `.в_ціле()`, `.корінь()`
- **Cast**: 13 реальних конвертацій через `як` (цл64↔дрб64↔тхт↔лог↔сим)
- **Макроси**: реальне збереження та виконання `макрос ім'я(пар) { тіло }`
- **Async/await**: `чекати вираз`, event loop з drain_async_queue
- **Dispatch ефектів**: `виконати_ефект("ефект", "операція", аргументи)`
- **Глобальні функції**: `мін()`, `макс()`, `абс()`
- **Імпорт**: пошук у робочій директорії + вкладених папках

### Виправлено
- Версії синхронізовано: 3.9.0 всюди
- Генератори мають per-instance стан (не глобальний VM буфер)
- Pipeline каррінг через CurriedBuiltin

### Тести: 80 Тризуб + 45 Rust = 125

## [3.6.0] - 2026-03-26

### Нові типи даних
- **Словники**: `словник("ключ", значення)` з методами `.довжина()`, `.ключі()`, `.значення()`, `.містить_ключ()`, `.отримати()`, `.додати()`, `.видалити()`
- **Множини**: `множина(1, 2, 3)` з методами `.містить()`, `.перетин()`, `.об'єднання()`, `.різниця()`

### Оператор `в` (belonging)
- `5 в [1,2,3,4,5]` — масив
- `3 в множина(1,2,3)` — множина
- `"ключ" в словник` — словник
- `5 в 1..10` / `10 в 1..=10` — діапазон
- `"світ" в "привіт світ"` — підрядок

### For-in розширення
- Ітерація по словниках: `для (пара в словник)`
- Ітерація по множинах: `для (елемент в множина)`
- Ітерація по рядках: `для (символ в "текст")`

### Pipeline каррінг (виправлено)
- `масив |> фільтрувати(|х| х > 2) |> перетворити(|х| х * 10)` — працює
- CurriedBuiltin замість зламаного Function з порожнім body

### Розширений REPL
- `:час <код>` — виміряти час виконання
- `:завантажити "файл"` — завантажити .тризуб файл

### Виправлення
- Порівняння масивів, кортежів (values_equal рекурсивно)
- 5 зламаних builtins реалізовано (сортувати, обернути, додати, діапазон)
- clap version синхронізовано

### Тести: 56 на Тризуб + 45 Rust = 101

## [3.5.0] - 2026-03-26

### Реалізовано у VM (реально працює)

- **Побітові оператори**: `<<`, `>>`, `^`, `~` працюють для цілих чисел
- **Контракти**: `вимагає { }` перевіряється ДО виклику, `гарантує(р) { }` ПІСЛЯ
  - При порушенні контракту — помилка "Контракт порушено"
- **Тест-раннер**: команда `тризуб тестувати файл.тризуб`
  - Знаходить `тест "назва" { }` блоки та запускає їх ізольовано
  - Підрахунок пройдених/провалених
- **Assert**: `перевірити вираз` — зупиняє виконання при хиба
- **Unsafe блоки**: `небезпечний { }` — виконуються без обмежень
- **Comptime блоки**: `компчас { }` — виконуються як звичайний код

### Нові ключові слова (лексер + парсер)

- Ефекти: `чистий`, `ефект`, `з_обробником`
- Контракти: `вимагає`, `гарантує`, `старе`, `інваріант`
- Тестування: `тест`, `фаз`, `бенчмарк`, `перевірити`, `виміряти`
- Макроси: `макрос`, `компчас`
- Системне: `небезпечний`, `асемблер`, `зовнішній`, `мінливий`, `упакований`, `переривання`
- Пам'ять: `персистентний`, `летючий`, `стійкий`, `спільний`
- Security: `можливість`, `дозвіл`, `пісочниця`
- UI: `вікно`, `стовпець`, `сітка`, `кнопка`
- Типи: `чсрозм`, `цлрозм`, `вказівник`, `пусто`

### Стандартна бібліотека (специфікації)

- `stdlib/ядро/персистентність.тризуб` — персистентна пам'ять
- `stdlib/ядро/безпека.тризуб` — capability-based security
- `stdlib/ядро/планувальник.тризуб` — планувальник з ігровим режимом
- `stdlib/ядро/інтерфейс.тризуб` — декларативний UI
- `stdlib/ядро/віртуалізація.тризуб` — запуск Windows/Linux програм
- `stdlib/json.тризуб` — JSON парсер/серіалізатор

### Тести

- 14 тестів на мові Тризуб (examples/тести_мови.тризуб)
- 45 Rust unit/integration тестів

## [2.0.0] - 2025-10-25

### Нові можливості мови

- Алгебраїчні типи даних (`тип Назва { Варіант1, Варіант2(Т) }`)
- Pattern matching (`зіставити` з guard-умовами, wildcard, деструктуризацією)
- Трейти (`трейт` + `реалізація ... для ...`)
- Pipeline оператор `|>`
- Лямбда-функції `|x| вираз`
- Оператор `?` для поширення помилок
- Інтерполяція рядків `ф"...{вираз}..."`
- Діапазони `1..10`, `1..=10`
- For-in цикли `для (х в колекція)`
- Вбудовані типи Опція та Результат
- Неявний return з функцій
- Try/catch/finally

### Стандартна бібліотека

- `колекції` — Вектор, Словник, Множина, Стек, Черга
- `час` — Дата, Тривалість, Хронометр
