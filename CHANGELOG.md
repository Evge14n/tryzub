## [2.0.0] - 2025-10-25

### Нові можливості мови

- **Алгебраїчні типи даних (enum)** — ключове слово `тип` для створення sum types з варіантами, що можуть містити дані
- **Зіставлення зразків (pattern matching)** — вираз `зіставити` з підтримкою деструктуризації, guard-умов (`якщо`), wildcard (`_`) та вкладених зразків
- **Трейти (типажі)** — ключове слово `трейт` для визначення інтерфейсів з методами за замовчуванням; `реалізація ... для ...` для імплементації
- **Оператор конвеєра `|>`** — передача результату попереднього виразу як першого аргументу наступної функції
- **Лямбда-функції** — синтаксис `|параметри| вираз` для анонімних функцій з захопленням замикання
- **Оператор `?` для поширення помилок** — автоматичне повернення помилки з функцій що повертають `Результат`
- **Інтерполяція рядків** — префікс `ф"Текст {вираз}"` для вбудовування виразів у рядки
- **Деструктуризація** — розпакування структур, масивів та кортежів у змінні
- **Діапазони** — синтаксис `1..10` (виключний) та `1..=10` (включний), ітерування `для (і в діапазон)`
- **Кортежі** — `(значення1, значення2)` як легковагі групування даних
- **Вбудовані типи `Опція<Т>` та `Результат<Т, П>`** — стандартні алгебраїчні типи для обробки відсутності значень та помилок
- **Ключове слово `себе`** — для посилання на поточний екземпляр в методах
- **Ключове слово `в`** — для ітерації по колекціях та діапазонах

### Покращення компілятора

- Повна реалізація VM для всіх нових конструкцій
- Покращена обробка помилок з вказівкою рядка та стовпця
- Підтримка вкладених коментарів `/* /* */ */`
- Розширена система типів з генериками

### Стандартна бібліотека

- Новий модуль `колекції` — Вектор, Словник, Множина, Черга, Стек
- Новий модуль `час` — робота з датою, часом, таймерами
- Вбудовані типи `Опція` та `Результат` з набором методів
- Розширено модуль `рядки` — підтримка інтерполяції, regex
- Розширено модуль `математика` — лінійна алгебра, комплексні числа

### Приклади

- Оновлено всі існуючі приклади до синтаксису v2
- Додано приклади з pattern matching, трейтами, pipeline
- Додано приклад банківської системи з трейтами та `Результат`

## [1.0.0] - 2025-04-06

### Перший офіційний реліз

**Автор:** Мартинюк Євген

### Додано
- Лексичний аналізатор з підтримкою українського синтаксису
- Синтаксичний аналізатор для базових конструкцій
- Віртуальна машина для інтерпретації коду
- Базова інтеграція з LLVM для компіляції
- Runtime система з керуванням пам'яттю
- Стандартна бібліотека: `математика`, `рядки`, `файли`
- CLI інтерфейс: `запустити`, `компілювати`, `перевірити`, `новий`, `форматувати`
- GitHub Actions для CI/CD
- Типи даних: цл8-64, чс8-64, дрб32-64, тхт, сим, лог
- Мовні конструкції: змінні, функції, if/else, for, while
- Приклади програм та документація

### Відомі обмеження
- Компіляція в нативний код через LLVM — експериментальна
- Немає алгебраїчних типів, трейтів, pattern matching
- Немає пакетного менеджера
- Обмежена стандартна бібліотека
